Receipe API Documentation

Endpoint Description Request Type
url/getAll Get all Receipes GET
url/getAll?page=1&sortBy=foodname&orderBy=desc&size=5 The Api Request Have several Sorting Methods.
Method are as Follows: sortBy, page, size, orderBy.
The Methods are Request Parameters		 GET
url/getAll/{category} The Api Can Sort the Receipes According to the category:
Breakfast, Lunch, Dinner and Dessert. 		GET
url/addReceipe The Api Can Add Receipes. As the form data. The data should be added in this format
id, foodname, description, ingredient, preparation, history, category( Breakfast, Lunch, Dinner and Dessert.),
type(Vegetarian and Non-vegetarian), image(File and Only Upload Images). 		POST
url/updatereceipeById/{id} The Receipes of Api Can be Update By Id. PUT
